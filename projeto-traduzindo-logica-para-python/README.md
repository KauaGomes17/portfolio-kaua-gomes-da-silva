# 🛠️ Python Utility Suite: Automação de Lógica de Negócios

## 📝 Descrição do Projeto
Este repositório contém uma coleção de scripts Python desenvolvidos para automatizar cálculos e processos de tomada de decisão em diferentes contextos operacionais. O foco principal é fornecer ferramentas interativas que facilitam a gestão de dados financeiros, acadêmicos e logísticos através de lógica de programação robusta.

O projeto foi estruturado para demonstrar o uso de estruturas de repetição, condicionais e manipulação de tipos de dados para resolver problemas do mundo real, como planejamento de investimentos e análise de métricas de desempenho.

## 📁 Estrutura do Projeto
O conjunto de ferramentas é composto por quatro módulos principais:

*   **Simulador de Poupança:** Calcula a evolução de investimentos com aportes mensais variáveis e metas de saldo[cite: 1].
*   **Sistema de Notas:** Automatiza o cálculo de médias acadêmicas e define o status de aprovação de turmas[cite: 2].
*   **Analisador de Clima:** Monitora temperaturas semanais e dispara alertas para condições extremas[cite: 3].
*   **Processador de Vendas:** Gerencia PDV simples com aplicação de descontos progressivos (até 10%) baseados no volume de compra[cite: 4].

## 🚀 Tecnologias Utilizadas
*   **Linguagem:** Python 3.10+
*   **Ambiente de Desenvolvimento:** Google Colab / Jupyter Notebook
*   **Paradigma:** Programação Estruturada

## 📊 Funcionalidades e Regras de Negócio
O sistema implementa validações críticas para garantir a integridade dos dados:
*   **Gestão Financeira:** Identificação automática do mês em que o usuário atinge a meta de R$ 10.000,00[cite: 1].
*   **Alertas Climáticos:** Notificação de "Condições Perigosas" para temperaturas acima de 45°C ou abaixo de -5°C[cite: 3].
*   **Logística de Vendas:** Cálculo automático de subtotal e aplicação de 10% de desconto para compras acima de R$ 500,00[cite: 4].
*   **Critérios Acadêmicos:** Classificação de alunos entre Aprovado (Média ≥ 7.0), Recuperação (Média ≥ 5.0) e Reprovado[cite: 2].
